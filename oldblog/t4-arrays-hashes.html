Arrays versus Hashes
Arrays Hashes 

  RossandRachel = ["We","were","on","a","BREAK!"]

  • From the example above, you can see that this iconic string would not really make sense if it were out of order.
  • Arrays have a specific order to them and the first index begins with 0(zero)
            array_name = [0,1,2,3,4]

So as you can see, even though an array can house 5 items, but its index only goes up to 4 because the count starts with zero rather than one.
Only these indexes are used to identify a specific position in an array: 

            RossandRachel[0]= "We"
            RossandRachel[1]= "were"
            RossandRachel[2]= "on"
            RossandRachel[3]= "a"
            RossandRachel[4]= "BREAK!"

There's so much you can do with arrays, but here's a little taste if you want to check how many items/elements are in an array:

            RossandRachel.length
            =>  5
          

           Friends = {

  "Rachel" => "Left a man at the alter"
  "Ross" => "Married a lesbian"
  "Joey" => "Threw a girl's wooden leg in a fire"
  "Phoebe" => "Fell in love with a gay ice dancer"
  "Chandler" => "Lived in a box"
  "Monica" => "Dated her ex's son"
            }

  • Hashes don't care about order, they act on a key-value basis, shown in the example above.
  • The keys are the names of the main 6 characters on the show, in no particular order.
  • The values that are paired with the characters are the things they've done that they aren't necessarily proud of...

Since order doesn't matter in hashes, you'd only need to know and call a key (a name in this case if you were curious about any of the Friends vices) to pull the related content. This is a great tool to use when devising an address book (ex. key= name; value = address). Or keep a list of gift ideas for each friend or.......... even a list of all the things (money, borrowed DVDs, games, etc.) each friend owes you over the years...hmmm 

      Friends["Ross"]
      ==> "Married a lesbian"
      Friends["Phoebe"]
      ==> "Fell in love with a gay ice dancer"
      Friends["Joey"]
      ==> "Threw a girl's wooden leg in a fire"
